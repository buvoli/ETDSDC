==================================================================================================

	This directory contains Fortran and MATLAB Code for 
	Buvoli T. "ETD Spectral Deferred Correction Methods", 2014
	Submitted to SINUM.

	All code is released under the MIT license (See LICENCE).

==================================================================================================

Directory Structure: 

	MATLAB 	- Contains MATLAB implementations of ETDSDC, IMEXSDC, and ETDRK4.

	Fortran - Contains fortran implementations of ETDSDC, IMEXSDC, and ETDRK4.
		  Requires FFTW, BLAS, LAPACK for compilation.

See README.txt files in individual directories for more specific information.
